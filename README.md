# HorseRacingPrediction

## About This Project
This is a simple project about determining the winner of a horse race. It uses Decision Trees to learn.
This project also uses XGBoost's learning-to-rank algorithms to rank the horses.

## Requirements
- Python 3.10+

## Installation
1. Clone the repository:
```bash
git clone https://github.com/tcan05/HorseRacingPrediction.git
```

To run the program you need to manually uncomment the parts you want use.
